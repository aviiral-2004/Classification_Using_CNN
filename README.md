Cat/Dog Image Classifier using CNN

This is a simple deep learning project using **TensorFlow** and **Flask** that classifies uploaded images as either a **CAT** or a **DOG**.
For Training and Testing, kaggale "cat/dog_classifier" is used
Features :-
- Upload any image using a web browser
- The model will classify it as a cat or a dog
- Built with a custom-trained Convolutional Neural Network (CNN)
- Simple Flask-based web UI (too simple)
